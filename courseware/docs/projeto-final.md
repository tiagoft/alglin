# Projeto Final

## Descrição

O projeto final tem como objetivo demonstrar que você é capaz de identificar um problema do mundo real, modelá-lo usando ferramentas de álgebra linear, e interpretar os resultados obtidos de volta no contexto do problema original.

## Etapas

1. **Definir um escopo.** Escolha um problema do mundo real. Modele-o como um problema de álgebra linear e, ao fim, retorne os resultados para o contexto do problema original.

2. **Encontrar ao menos uma referência** (blog, vídeo, artigo, etc.) que já tenha realizado trabalho semelhante.

3. **Implementar a solução** e interpretar os resultados frente a dados reais e ao problema real.

4. **Escrever um relatório** — formato: 1 página, formato IEEE. Referências podem aparecer na página 2, se necessário.

5. **Submeter o relatório em PDF.** O grupo recebe nota pelo relatório. Após a entrega, uma arguição individual determina a nota individual. A nota total é a média geométrica das duas.

## Avaliação

O projeto é avaliado em duas dimensões, cuja média geométrica compõe a nota final:

* **Nota de grupo:** qualidade do relatório entregue (conteúdo completo, linguagem adequada, conclusões sustentadas por fatos, referência abordada criticamente).
* **Nota individual:** arguição individual, na qual cada aluno defende suas decisões de projeto.

Na arguição, espera-se que o aluno seja capaz de responder:

* Por que a solução escolhida foi aquela (frente a outras alternativas)?
* Como ela se compara com a sua referência?
* Quais são as limitações da sua solução, e quais dados ou experimentos as evidenciam (exemplo: assumir que um fenômeno é linear, quando ele na verdade não é - como isso impacta o resultado final)?
* Quais são as implicações dos resultados para o problema real?
